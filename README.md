# BetterGeometry V0.1.0.4

BetterGeometry is a BetterTools ChatGPT app that gives geometry explanations a visual layer.

## Versioning

`Vpublish.feature.subfeature.patch`

Current version: `V0.1.0.4`

Every visible app/widget must show the current version in a corner.

## This version

Patch update for the V0.1.0.0 generic renderer:

- Replaces the crowded default demo with a cleaner scene.
- Separates triangle and circle labels so they no longer collide.
- Reduces text outline weight and point-label size.
- Improves segment label placement.
- Keeps the ChatGPT widget scroll-free.
- Bumps the widget URI to `ui://widget/bettergeometry-v0.1.0.4.html`.
